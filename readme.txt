SISTEM PERPUSTAKAAN DIGITAL
Project UAS Struktur Data dan Algoritma Menggunakan bahasa pemrograman Java.
=

CARA MENJALANKAN PROGRAM
- Buka Folder Tubes Alpro
- Buka File TugasAkhirAlpro.java
- Code Siap Di Run
- ! NOTE JANGAN LUPA BUAT FOLDER BER-ISIKAN FILE Data.txt DALAM FOLDER TUBES ALPRO !

Program ini merupakan sistem perpustakaan digital berbasis java
Struktur Data Dan Algoritmanya Ber-isikan tentang :
- Array Statis
- Class/Object
- Boolean Flag
- CRUD
- Bubble Sort
- Selection Sort
- Linear Search
- Binary Search
- File Handling

============================================

Pada Bagian Awal Run Terdapa 2 Opsi Yaitu:
- Admin
- User

============================================

Pada Bagian ADMIN Terdapat Menu
- Tambah Buku
- Tampilkan Buku
- Edit Buku
- Hapus Buku
- Sorting ID (Bubble Sort)
- Sorting Judul (Selection Sort)
- Kembali (Kembali Ke Menu Run Awal)

============================================

Pada Bagian USER Terdapat Menu
- Tampilkan Buku
- Pinjam Buku
- Kembalikan Buku
- Cari Buku (Judul - Linear)
- Cari Buku (ID - Binary)
- Buku dengan Stok Terbanyak
- Cari Berdasarkan Kategori
- Kembali
